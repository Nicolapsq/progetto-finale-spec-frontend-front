Sviluppo di un e-commerce responsive per la vendita di laptop.
Il progetto include:
- Una homepage con lista prodotti, una barra di ricerca per nome e una barra per selezionare una categotia,
- Una pagina per i prodotti preferiti,
- Una pagina del singolo prodotto,
- Una pagina che confronta due prodotti
🛠️ Stack: React
